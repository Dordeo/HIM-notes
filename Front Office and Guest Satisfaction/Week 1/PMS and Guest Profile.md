[[PMS (Property Management System)]]  - A Property Management System (PMS) is a software platform that helps manage the day-to-day operations of properties, including reservations, maintenance, finances, and communication. Check out the newHotel framework.

[[ Guest Profile ]] - a database that contains information about their history , travelling agent , preferences.


# PMS core functionality :

- Reservations & Rate Management
- Front Desk & Profiles
- Rooms Management
- Cashiering
- Accounts Receivable & Commissions
- Reporting & Analytics

## Interfaces : Integrated Systems

- Call Accounting Systems -  track calls, automates billing and manages the telecommunication.
- Energy Management Systems - increases efficiency
- Video on demand - television and series of entertainment
- F&B Outlets - Food and bars
- Electronic Locking System - secure access control
- Minibar - automated F&B
-  POS - point-of-say, service and the room account
 
## Affordable & Clean Energy: SDG7
**Sustainable Development Goals**
- Ways to reduce costs by using replenishable energy or other structural demands
- Water reusability
- Temperature differential is monitored
- Minimization on the negative environmental impact

# [[Guest Cycle]]

### Pre arrival
Reservations and pre - payment, guest profile  setup
### Arrival
Registration, room assignment, key issuance , credit establishment, welcome services
### Occupancy
Services, charge postings, information assistance,  room status updates
### Departure
Account settlement, feedback, express check out option


# Room types

Room types are divided by multiple categories

| Occupancy | Definition |
| --------- | ---------- |
| Single    | 1          |
| Double    | 2          |
| Triple    | 3          |
| Quad      | 4          |

| Bed Configuration | Definition              |
| ----------------- | ----------------------- |
| Single            | 1 normal bed            |
| Twin              | 2 normal beds           |
| Double            | 1 big bed for 2 people  |
| King              | 1 kings size bed        |
| Queen             | 1 queen size bed        |
| Double-Double     | 2 big beds for 2 people |

| Category     | Definition                                 |
| ------------ | ------------------------------------------ |
| Studio       | Combined living and sleeping area          |
| Junior Suite | Big sitting are inside a room              |
| Suite        | Separate sleeping and sitting areas        |
| Barrier Free | ADA accessibility compliant(for dissabled) |

| Room Arrengment | Definition                |
| --------------- | ------------------------- |
| Connecting      | A door between the rooms  |
| Adjoining       | A wall between the rooms  |
| Adjacent        | On the same floor, nearby |

# Room status


| Room status      | Definition                               |
| ---------------- | ---------------------------------------- |
| Occupied         | Currently in use                         |
| Vacant and ready | Clean, inspected room avaible for sale   |
| On-Change        | Room being cleaned or prepared           |
| Stayover         | Guest staying over night                 |
| Due Out          | Guest scheduled to check out today       |
| Do Not Disturb   | Guest has requested privacy              |
| Out-of-Order     | Room renderedunviable due to maintenance |
| Out-of-Service   | Room under renovation                    |
| Lock-Out         | Guest access restricted                  |
| Sleep-Out        | Guest slept elsewhere                    |
| Skipper          | Guest departed without checkout          |
| Sleeper          | Checked out guest showing as in-house    |
